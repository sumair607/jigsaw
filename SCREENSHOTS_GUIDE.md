## Using Screenshots for App Store

1. **Go to Google Play Console**
   - Navigate to your app's store listing
   - Select "Graphic Assets" section

2. **Upload Screenshots**
   - Upload all 5 screenshots in order

## Best Practices for App Store Screenshots

### Design Principles

- **First Screenshot:** Make it count - shows home screen with key features
- **Progression:** Show the user journey from start to completion
- **Clarity:** Use large text and clear visuals
- **Branding:** Consistent use of app colors and logo
- **Engagement:** Include celebratory moments and rewards

### What Works Well

‚úÖ **Do:**
- Show real gameplay, not just menus
- Use high-quality, vibrant images
- Include text overlays for key features
- Show progression and rewards
- Use consistent branding

‚ùå **Don't:**
- Use blurry or low-quality images
- Overload with too much text
- Show error states or bugs
- Use placeholder content
- Make text too small to read

### Optimization Tips

1. **Text Overlay:** Keep text minimal and readable
2. **Color Contrast:** Ensure text stands out against backgrounds
3. **Consistency:** Use same device frame and theme across all screenshots
4. **Localization:** Translate text for different markets
5. **A/B Testing:** Test different screenshot orders to see which converts better
